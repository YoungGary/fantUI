# vue-fantui

## Install

```
npm install vue-fantui -S
```

## Quick Start

```js
import Vue from "vue"
import fantui from "vue-fantui"

Vue.use(fantui)

// or
import {
	Icon,
	Button,
	// ...
} from "vue-fantui"

Vue.component(Icon.name, Icon)
Vue.component(Button.name, Button)
```

## styles

```js
import "vue-fantui/style/index.scss"
```

## Browser Support

Modern browsers and Internet Explorer 10+.
